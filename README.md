# CV Interactivo – Diego Nicolás Plaza Veliz

Currículum Vitae interactivo desarrollado como evaluación final del Módulo 2 del curso Full Stack JavaScript (Kibernum / Talento Digital Chile).

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- jQuery
- Git y GitHub

## Características

- Diseño responsivo con Bootstrap.
- Navbar para navegar entre las secciones del CV.
- Botón para cambiar entre tema claro y oscuro.
- Animación de aparición de secciones al hacer scroll.
- Formulario de contacto con validación en tiempo real.
- Botón para cambiar los colores de la sección de habilidades.
